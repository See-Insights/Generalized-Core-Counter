Detailed Description

Persistent Data Storage Structures - EEPROM/Retained Memory Management.

Defines data structures for device configuration, sensor settings, and runtime state that persist across power cycles and reboots. Uses StorageHelperRK for efficient EEPROM/retained memory operations with automatic validation and versioning.

Counting mode defines how the device processes sensor events.

COUNTING: Interrupt-driven event counting. OCCUPANCY: Interrupt-driven occupied/unoccupied tracking. SCHEDULED: Non-interrupt, time-based sampling.
